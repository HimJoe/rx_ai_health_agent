Overview

Rx-AI is a Retrieval-Augmented Health Agent that integrates large language models with a curated medical knowledge base and dynamic user profiles (FHIR-inspired) to deliver personalized, evidence-grounded health recommendations. It implements safety checks and explicit disclaimers to ensure responsible use.

Download the full paper (PDF)

System Architecture

The core components:

  • Health Knowledge Base: Vector store (FAISS/Pinecone)
  • User Profile Manager: FHIR-style user data
  • RAG Engine: Embedding, retrieval, prompt assembly
  • Safety Module: Emergency detection, disclaimers
